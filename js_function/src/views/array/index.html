数字

/^[0-9]*$/g

汉字

/^[\u4e00-\u9fa5]{0,}$/g

身份证号

(15位、18位数字)，最后一位是校验位，可能为数字或字符X 
/(^\d{15}$)|(^\d{18}$)|(^\d{17}(\d|X|x)$)/g

IPv4地址

/((2(5[0-5]|[0-4]\d))|[0-1]?\d{1,2})(\.((2(5[0-5]|[0-4]\d))|[0-1]?\d{1,2})){3}/g

中国邮政编码

/[1-9]\d{5}(?!\d) (中国邮政编码为6位数字)/g

电话号码
